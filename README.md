## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Abilon/csvParser.git
   cd csvParser
2. Установка зависимостей
    ```
    npm install
3. Запуск тестов:
    ```
    npm test
4. Запуск линта:
    ```
    npm run lint
5. Запуск автоформатирования:
    ```
    npm run format
6. Запустите сборку для разработки:
    ```
    npm start
7. Запуск сборки в режиме продакшена:
    ```
    npm build
